🌟 МУТТАҲИД СОХТАНИ ЯКЧАНД МАССИВ ДАР JAVASCRIPT
📖 Муқаддима
Дар забони барномасозии JavaScript бисёр вақт лозим меояд, ки маълумотҳои якчанд массивро ба як массив ҷамъ кунем. Барои иҷрои ин вазифа метавон аз даврҳо (for...of) ва методи push() истифода бурд.
Мисоли зер маҳз ҳамин амалро иҷро мекунад: ҳамаи элементҳои якчанд массивро гирифта, дар як массиви нав ҷойгир менамояд.

💻 Код
function main(...rest) {    let newArr = [];    for (let elemOfArr of rest) {        for (const elem of elemOfArr) {            newArr.push(elem);        }    }    return newArr;}console.log(main([1,2,3],[4,5,6],[7,8]));

⚙️ Тарзи кори барнома
Функсия бо истифода аз ...rest ҳамаи массивҳои воридшударо қабул мекунад. Баъдан массиви нави холӣ бо номи newArr сохта мешавад.
Бо ёрии даври якуми for...of ҳар як массив алоҳида гирифта мешавад. Сипас даври дуюм ҳамаи элементҳои дохили он массивро мегузарад ва онҳоро тавассути push() ба массиви нав илова мекунад.
Дар охир функсия массиви пурра муттаҳидшударо бармегардонад.

📥 Воридот
main([1,2,3],[4,5,6],[7,8])
📤 Баромад
[1,2,3,4,5,6,7,8]

🎯 Афзалиятҳои ин усул
✔️ Код сода ва фаҳмо аст.
✔️ Барои ҳар қадар массив кор мекунад.
✔️ Хондан ва нигоҳдории код осон мебошад.
✔️ Барои омӯзиши массивҳо ва даврҳо бисёр муфид аст.

🏁 Хулоса
Ин барнома намунаи хуби кор бо массивҳо дар JavaScript мебошад. Бо истифода аз Rest Parameter, даврҳои for...of ва методи push() мо метавонем якчанд массивро ба як массиви ягона муттаҳид намоем. Ин усул ҳам барои донишҷӯён ва ҳам барои барномасозони навкор фаҳмо ва самаранок мебошад.
✨ Натиҷаи ниҳоӣ:
[1,2,3,4,5,6,7,8]

«Содагӣ яке аз нишонаҳои барномасозии хуб аст. Ин мисол бо коди кӯтоҳ ва равшан масъалаи муттаҳид кардани массивҳоро ба таври самаранок ҳал мекунад.» 🚀📚



function main(...rest) {
    let newArr = [];

    for (let elemOfArr of rest) {
        for (const elem of elemOfArr) {
            newArr.push(elem);
        }
    }

    return newArr;
}

console.log(main([1,2,3],[4,5,6],[7,8]));


🌟 DESCTRUCTURING ВА SPREAD OPERATOR ДАР JAVASCRIPT 🌟
📖 Муқаддима

Дар JavaScript яке аз имкониятҳои хеле муфид ва замонавӣ Destructuring ва Spread Operator (...) мебошад. Ин имкониятҳо ба барномасоз кӯмак мекунанд, ки кодро кӯтоҳтар, фаҳмотар ва зеботар нависад.

🔹 Коди мисол 


// destructuring

let [a, b, ...arr2] = arr;

console.log(arr2);


// spread

let max = Math.max(...arr);

console.log(...arr);


🎯 Хулоса
Дар ин мисол ду имконияти муҳими JavaScript истифода шудааст:
✅ Destructuring — элементҳои массивро ҷудо ва ба тағйирёбандаҳо ҷойгир мекунад.
✅ Spread Operator — элементҳои массивро паҳн намуда, онҳоро ҳамчун қиматҳои алоҳида истифода мебарад.
Ин имкониятҳо кодро кӯтоҳ, зебо, фаҳмо ва касбӣ мегардонанд ва имрӯз дар аксари лоиҳаҳои JavaScript васеъ истифода мешаванд.

✨ Destructuring маълумотро ҷудо мекунад, Spread бошад маълумотро паҳн мекунад. ✨
